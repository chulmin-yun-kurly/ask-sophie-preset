### 1. 구매 가이드
* 정의: 특정 조건 없이 카테고리 전반에 대한 추천·선택 기준. "뭘 사야 해?"가 핵심 의도.
* 포함 요소: 입문자 가이드, 선택 기준 개요.
* 경계 조건:
  * 특정 용도 지정 시 → 용도/상황
  * 특정 스펙 조건 시 → 명시적 정보 기반
  * 특정 상품 지목 시 → 개별 상품 인트로(답변 유형)

### 2. 컬리 특화
* 정의: 컬리 MD 큐레이션, 산지 방문, 단독 상품 등 컬리만 줄 수 있는 정보.
* 포함 요소: MD 큐레이션 스토리, 산지 방문기, 컬리 단독/한정 상품, 상품위원회 검증 과정 등.
* 경계 조건:
  * 품질 스펙이 핵심 → 품질/인증
  * "가장 많이 팔리는" → 사회적 증거

### 3. 품질/인증
* 정의: 정량적 품질 지표 또는 공인 인증. 숫자와 라벨이 판단 근거.
* 포함 요소: 산도, 폴리페놀, 발연점, 등급, DOP/IGP/유기농/SIQEV, 수확 연도, 착유 시간, 냉압착 여부 등.
* 경계 조건:
  * 감각적 묘사 → 정성적 특성
  * "왜 비싸요" → 차별점

### 4. 정성적 특성 (맛/향)
* 정의: 맛, 향, 텍스처 등 정성적 특성 기준.
* 포함 요소: 풍미 프로필 (풀향, 과일향, 너티, 스파이시 등), 쓴맛/매운맛/단맛 강도, 텍스처, 후미 (페퍼리, 아몬드 등), 색감 등.
* 경계 조건:
  * 숫자 기준(산도 등) → 품질/인증
  * 특정 요리와의 조합 → 페어링

### 5. 용도/상황
* 정의: 사용 맥락(어디에/어떻게 쓸 것인가)
* 포함 요소: 조리 용도, 섭취 방식, 사용 상황, 식단 맥락 등.
* 경계 조건:
  * 특정 요리명 매칭 → 페어링
  * 사용자 속성 기준 → 타겟 고객
  * 패키지 형태 조건 → 명시적 정보 기반

### 6. 타겟 고객
* 정의: 사용자의 속성(연령, 경험, 상황 등)을 기준으로 "이 상품이 나한테 맞는가" 관련.
* 포함 요소: 연령/생애주기 (유아, 어린이, 임산부, 시니어), 경험 수준 (입문자, 마니아), 라이프스타일 (건강 관리, 미식 탐구), 특수 조건 (알레르기, 식이 제한) 등.
* 경계 조건:
  * 섭취 방식 질문 → 용도/상황
  * 선물용 → 용도/상황

### 7. 페어링 (음식)
* 정의: 특정 요리·식재료와의 조합.
* 포함 요소: 요리 매칭, 식재료 조합, 의외의 페어링 등.
* 경계 조건:
  * 조리법 질문 → 레시피
  * 넓은 조리 용도("샐러드용") → 용도/상황

### 8. 명시적 정보 기반
* 정의: 상품의 객관적·구조화된 스펙. 필터링 가능한 정보 여부가 기준.
* 포함 요소: 가격/가격대, 용량, 패키지 형태, 원산지, 브랜드명 등.
* 경계 조건:
  * 품질 수치 → 품질/인증
  * 할인/쿠폰 → 실시간 API 영역

### 9. 사회적 증거
* 정의: 다른 사용자의 행동·평가 관련. "다른 사람들은 뭘 사?" "얼마나 인기 있어?"가 핵심 의도.
* 포함 요소: 판매량, 후기 수/평점, 재구매율, 실시간 조회 수, 고객 세그먼트별 인기 (VIP 구매, 3040 인기), 전문가 추천 등.
* 경계 조건:
  * MD 추천 → 컬리 특화
  * 수상 경력 → 품질/인증

### 10. 차별점
* 정의: "왜 다른가", "왜 비싼가", "뭐가 특별한가"가 핵심 의도. 상품 간 또는 등급 간 차이의 근거가 핵심.
* 포함 요소: 등급별 차이, 제조 방식 차이, 가격 차이 근거, 동일 카테고리 내 상대적 우위, 일반 상품 대비 프리미엄의 이유 등.
* 경계 조건:
  * A vs B 병렬 비교 → 비교(답변 유형)

### 11. 트리비아 (잡학)
* 정의: 구매 결정과 직접 연결되지 않지만, 카테고리에 대한 흥미·이해를 높이는 배경 지식. "알아두면 재밌는" 정보.
* 포함 요소: 역사, 문화, 재미 요소, 시장/트렌드 등.
* 경계 조건:
  * 건강 효능 주장 → 삭제
  * 실용적 보관 정보 → 보관/관리
  * "왜 좋아요" → 차별점

### 12. 보관/관리
* 정의: 구매 후 보관·관리·사용 기한에 대한 실용 정보.
* 포함 요소: 보관 온도/장소 (냉장 여부, 직사광선 차단), 산화 방지법, 개봉 후 사용 기한, 유통기한 해석, 변질 징후 확인법, 용기별 보관 차이 등.
* 경계 조건:
  * 배송 중 품질 → 배송

### 13. 레시피
* 정의: 상품을 활용한 구체적 조리법·만드는 방법. "어떻게 만들어?"가 핵심 의도.
* 포함 요소: 조리 순서/방법, 재료 비율, 활용 팁, 이색 레시피 등.
* 경계 조건:
  * 음식 매칭 추천 → 페어링
  * 넓은 조리 용도 → 용도/상황

### 14. 배송
* 정의: 배송 과정·포장 상태·배송 중 품질 보존에 대한 정보.
* 포함 요소: 배송 방식 (상온/냉장), 포장 안전성 (파손 방지), 계절별 품질 영향, 배송 소요 시간 등.
* 경계 조건:
  * 구매 후 보관 → 보관/관리
  * 배송 추적/일정 → AI 검색 범위 밖 (CS 영역)
